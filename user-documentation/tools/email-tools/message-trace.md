# Message Trace

Message Trace provides the ability for you to trace an e-mail instantly from to any recipient, or any sender over the last 10 days (max).

{% hint style="info" %}
Wildcard(\*) search is supported. Ex. "\*@contoso.com"
{% endhint %}
